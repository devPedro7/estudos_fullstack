BOX MODEL É UM CONTEÚDO INPORTANTE, POIS A PARTIR DELE NÓS CONSEGUIMOS ENTENDER COMO FUNCIONA
OS ESPAÇAMENTOS DOS ELEMENTOS DO NOSSO SITE.

MARGIN: É UM ESPAÇAMENTO EXTERNO. ELE COLOCA ESPAÇAMENTO EXTERNAMENTE AO REDOR DOS NOSSOS ELEMENTOS.
 HÁ DIVERSAS VARIAÇÕES: 

    1º margin: auto - ELE ATRIBUI O MESMO ESPAÇAMENTO PARA AS LATERAIS DOS ELEMENTOS. USADO PARA
                      CENTRALIZAÇÃO DOS CONTEÚDOS.
    
    2º margin: top - ESPAÇAMENTO EXTERNO EM CIMA;
       margin: left - ESPAÇAMENTO EXTERNO NA ESQUERDA;
       margin: rigth - ESPAÇAMENTO EXTERNO NA DIREITA;
       margin: bottom - ESPAÇAMENTO EXTERNO EMBAIXO;

PADDING: É UM ESPAÇAMENTO INTERNO. ELE INSERE UM ESPAÇAMENTO INTERNAMENTE DENTRO DOS NOSSOS ELEMENTOS.
AGINDO DE FORMA CONTRÁRIA DO NOSSO MARGIN.

O PADDING POSSUI AS MESMAS PROPIEDADES DO MARGIN, EXCLUINDO O auto.

BORDER: É UMA BORDA QUE PODEMOS INSERIR NOS NOSSOS ELEMENTOS.


O BOX MODEL FUNCIONA: MARGIN > BORDER > PADDING > NOSSOS ELEMENTOS.
