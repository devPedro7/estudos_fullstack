-- PODEMOS TRABALHAR COM DATAS NO JAVASCRIPT.
PARA CRIAR DATAS FAZEMOS DA SEGUINTE FORMA: 
1º CRIAMOS UMA VARIÁVEL E ATRIBUIMOS A ELA UM 'new Date()' - DESSA FORMA ELE IRÁ PEGAR A DATA ATUAL DO SEU SISTEMA
    var data = new Date()
2º PODEMOS REALIZAR OPERAÇÕES COM ELA:
PEGAR A HORA getHour()
PEGAR OS MINUTOS getMinutes()
PEGAR OS SEGUNDOS getSeconds()
PEGAR OS MILISEGUNDOS getMiliseconds()
PEGAR O DIA getDate()
PEGAR O MES getMonth()
PEGAR O ANO getFullYear()
PEGAR O DIA DA SEMANA getDay() - RETORNA UMA INTEIRO 0 = DOMINGO E 6 = SÁBADO

APENAS TEMOS QUE INSERIR UM VALOR DENTRO DO 'Date("")'
