JAVASCRIPT COMANDOS:

-- PODEMOS UTILIZAR O JAVASCRIPT PARA DIVERSOS COMANDOS EM NOSSAS PÁGINAS:

1º ALERTA NA TELA: alert('mensagem')
2º SOLICITAR AO USUARIO INSERIR ALGO NA TELA: prompt('mensagem')


-- PODEMOS TAMBÉM MANIPULAR ELEMENTOS HTML COM O NOSSO JAVASCRIPT: PARA ISSO DEVEMOS UTILIZAR O "document"

Document: É A ÁREA INTEIRA DA PÁGINA.

1º COMANDO PARA ESCREVER COM UM ELEMENTO HTML: document.write("<h1>Teste</h1>") 
1.1º ESCREVENDO UMA MENSAGEM RECEBENDO UMA VARIÁVEL: document.write("<h2>Olá" +nome + "</h2>") 
ou document.write(`<h2>Olá ${nome}</h2>`)


-- PODEMOS CRIAR ELEMENTOS HTML COM O JAVASCRIPT : document.createElement("nome do elemento")

1º CRIANDO UM BOTÃO: document.createElement("button") - CRIA UM BOTAO + innerText CRIA UM TEXTO PARA O BOTAO
OBS - ISSO DEVE SER UMA VARIAVEL QUE RECEBE ESSE ATRIBUTO.
1.1º PARA INSERIR DENTRO DO DOCUMENTO HTML USAMOS: variavel + inner.appendChild(nome da variavel que recebe o
botão)


-- PODEMOS CRIAR UMA LISTA DE DADOS SEJA ELA DO TIPO QUE FOR, CONHECEMOS ESSA LISTA COMO: arrays.

--PARA DECLARAR UM ARRAY FAZEMOS DA SEGUINTE FORMA:
1º DECLARAMOS UMA VARIÁVEL, DAMOS UM NOME E FAZEMOS ELA RECEBER UM PAR DE CHAVES E OS ELEMENTOS QUE SERÃO OS
TIPOS DE DADOS QUE ELA VAI TER:

    EXEMPLO:  var nome = ['Pedro', 'Jakson', 'Catarina', 'Tabata'] - array de Strings.

-- EM ALGUMAS LINGUUAGENS DE PROGRAMAÇÃO O ARRAY DEVE COMEÇAR E TERMINAR COM O SEU TIPO DE DADO DEFINIDO, Strings
TERMINAM EM Strings, INTEIROS TERMINAM COM INTEIROS ETC. EM JAVASCRIPT TEMOS A FLEXIBILIDADE DE MESCLAR OS DADOS.


-- PODEMOS SOLICITAR PARA UMA LISTA SE TEM UM DETERMINADO ELEMENTO, E SE TIVER ELE RETORNA A POSIÇÃO:
lista.indexOf('Elemento') > retorna a posição. QUANDO ELE NÃO ENCONTRA UM ELEMENTO ELE RETORNA "-1"

-- PODEMOS ADICIONAR ELEMENTOS DE FORMA DINÂMICA NO FIM DA NOSSA LISTA: 
lista.push("elemento")

-- PODEMOS EXCLUIR O PRIMEIRO ELEMENTO DA NOSSA LISTA:
lista.shift() > ELE RETORNA O ELEMENTO QUE FOI REMOVIDO.

-- PODEMOS REMOVER O ÚLTIMO ELEMENTO DA LISTA:
lista.pop() > RETORNA O QUE ELE RETIROU.

-- PODEMOS INFORMAR DE QUE FORMA QUEREMOS SEPARAR OS ITENS DA NOSSA LISTA: 
lista.join("," ou ";" ....)

-- O PROMPT SEMPRE DEVOLVE O VALOR DE UMA STRING.
